# 上传到 GitHub

仓库地址：

```txt
https://github.com/ovo223/seven
```

## 需要上传的文件和目录

请上传当前项目根目录下的这些内容：

```txt
app/
components/
lib/
.env.example
.gitignore
DEPLOY.md
next-env.d.ts
next.config.mjs
package.json
postcss.config.mjs
preview.html
README.md
tailwind.config.ts
tsconfig.json
```

不要上传：

```txt
node_modules/
.next/
.env.local
```

## 网页上传方式

1. 打开仓库：

```txt
https://github.com/ovo223/seven
```

2. 点击页面里的：

```txt
uploading an existing file
```

3. 把上面列出的文件和目录拖进去。

4. Commit message 填：

```txt
Initial project upload
```

5. 点击：

```txt
Commit changes
```

## 上传完成后

上传完成后，可以去 Vercel 导入这个 GitHub 仓库部署。
